# Avatar

### Import

```jsx copy
import { VKAvatar } from "@vivakits/react-components";
```

### Variants

The Avatar component comes with 2 different variants that you can change using the `variant` prop.

<Tabs items={['Preview', 'Code']}>

  <Tabs.Tab>
export const AvatarVariantGroup = () => {
    const variants = [ "image" ,"text"];
    const imageSrc = "/avatar.jpg"

return (

<div className="flex gap-6">
  {variants.map((variant) => (
    <VKAvatar
      key={variant}
      variant={variant}
      size="md"
      color="primary"
      src={imageSrc}
    />
  ))}
</div>
); };

    <AvatarVariantGroup />

</Tabs.Tab>

  <Tabs.Tab>
```jsx copy

const imageSrc = "/avatar.jpg"

    <div className="flex gap-6">
        <VKAvatar variant="image" size="md" src={imageSrc}/>
        <VKAvatar variant="text" size="md" color="primary"/>
    </div>

````
</Tabs.Tab>
</Tabs>
