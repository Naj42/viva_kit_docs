### Import

```jsx copy
import { VKActionIcon } from "@vivakits/react-components";
```

### Variant

The ActionIcon component comes with 3 different variants that you can change using the `variant` prop.

<Tabs items={['Preview', 'Code']}>

  <Tabs.Tab>
