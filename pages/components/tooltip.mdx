# Tooltip

### Positions

The Tooltip component can be positioned in various ways using the `position` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipPositionGroup = () => {
  return (
    <div className="flex items-center gap-8 flex-wrap">
      <Tooltip position="top" content="Top Tooltip">Top</Tooltip>
      <Tooltip position="topLeft" content="Top Left Tooltip">Top Left</Tooltip> 
      <Tooltip position="topRight" content="Top Right Tooltip">Top Right</Tooltip>
      <Tooltip position="left" content="Left Tooltip">Left</Tooltip>
      <Tooltip position="bottom" content="Bottom Tooltip">Bottom</Tooltip>
      <Tooltip position="bottomLeft" content="Bottom Left Tooltip">Bottom Left</Tooltip>
      <Tooltip position="bottomRight" content="Bottom Right Tooltip">Bottom Right</Tooltip>
      <Tooltip position="right" content="Right Tooltip">Right</Tooltip>
    </div>
  );
};

<TooltipPositionGroup />
</Tabs.Tab>

### Content

The Tooltip component can display any content passed to it using the `content` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipContentGroup = () => {
  return (
    <Tooltip content="This is a tooltip">Hover me</Tooltip>
  );
};

<TooltipContentGroup />
</Tabs.Tab>

<Tabs.Tab>
```jsx copy
<Tooltip content="This is a tooltip">Hover me</Tooltip>
```
</Tabs.Tab>
</Tabs>

### Caret

The Tooltip component can display a caret pointing towards the content using the `caret` prop.

<Tabs.Tab>

    <div className="flex items-center gap-8 flex-wrap">

      <Tooltip caret={false} content="This is a tooltip without caret">Hover me</Tooltip>
    </div>
  );
};

<TooltipCaretGroup />
</Tabs.Tab>

### Sizes

The Tooltip component comes with 2 different sizes that you can change it using the `size` prop. The primary and secondary buttons can be used only in large tooltips. The width of small tooltips change according to the text length.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipSizeGroup = () => {
  return (
    <div className="flex items-end gap-3">
        <Tooltip size="sm" content="Small Tooltip">Small</Tooltip>
        <Tooltip size="lg" content="Large Tooltip">Large</Tooltip>
    </div>
  );
};

<TooltipSizeGroup />
</Tabs.Tab>

  <Tabs.Tab>
```jsx copy
<div className="flex items-end gap-3">
    <Tooltip size="sm" content="Small Tooltip">Small</Tooltip>
    <Tooltip size="lg" content="Large Tooltip">Large</Tooltip>
</div>
  ```
  </Tabs.Tab>
</Tabs>
