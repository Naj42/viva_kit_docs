# Tooltip

### Content

The Tooltip component can display any content passed to it using the `content` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipContentGroup = () => {
  return (
    <div className="flex space-x-1">
      <Tooltip content="This is a tooltip">
        <Button className="w-28" size="md" variant="light">Hover me</Button>
      </Tooltip>
      <Tooltip content={<img src="https://nextra.site/og.jpeg" className="w-48"></img>}>
        <Button className="w-28" size="md" variant="light">Hover me</Button>
      </Tooltip>
    </div>
  );
};

<TooltipContentGroup />
</Tabs.Tab>

### Caret

The Tooltip component allows you to toggle the caret on and off using the `caret` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipCaretGroup = () => {
  return (
    <div className="flex space-x-1">
      <Tooltip caret={true} content="This is a tooltip with caret">
        <Button className="w-28" size="md" variant="light">With caret</Button>
      </Tooltip>
      <Tooltip caret={false} content="This is a tooltip without caret">
        <Button className="w-32" size="md" variant="light">Without caret</Button>
      </Tooltip>
    </div>
  );
};

<TooltipCaretGroup />
</Tabs.Tab>

<Tabs.Tab>

```jsx copy
import React from 'react';

      </Tooltip>

        <Button className="w-32" size="md" variant="light">Without caret</Button>
```

### Close Button

The Tooltip component can display a close button using the `closeButton` prop.

export const TooltipCloseButtonGroup = () => {
  return (
    <Tooltip closeButton content="This is a tooltip" className="bg-white text-black text-xl" caretColor="#FFFFFF">
      <Button className="w-28" size="md" variant="light">Close</Button>
    </Tooltip>
  );
};

<TooltipCloseButtonGroup />

```jsx copy
  content="This is a tooltip"

    Close
  </Button>
```

### Max width

The Tooltip component can have a maximum width using the `maxWidth` prop. The tooltip will not resize beyond this width automatically.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipMaxWidthGroup = () => {
  return (
    <div className="flex space-x-1">
      <Tooltip content="This is a very very very very big tooltip">
        <Button className="w-28" size="md" variant="light">No fixed width</Button>
      </Tooltip>
      <Tooltip maxWidth="150px" content="This is a very very very very big tooltip">
        <Button className="w-28" size="md" variant="light">Fixed width to 150px</Button>
      </Tooltip>
    </div>
  );
};

<TooltipMaxWidthGroup />
</Tabs.Tab>

<Tabs.Tab>
