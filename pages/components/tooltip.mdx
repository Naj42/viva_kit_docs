# Tooltip

### Position

The Tooltip component can be positioned in various ways using the `position` prop.

      </Tooltip>
      <Tooltip position="topLeft" content="Top Left Tooltip">
        <Button className="w-28">Top Left</Button>
      </Tooltip> 
      <Tooltip position="topRight" content="Top Right Tooltip">
        <Button className="w-28">Top Right</Button>
      </Tooltip>
      <Tooltip position="bottom" content="Bottom Tooltip">
        <Button className="w-28">Bottomt</Button>
      </Tooltip>
      <Tooltip position="bottomLeft" content="Bottom Left Tooltip">
        <Button className="w-28">Bottomt Left</Button>
      </Tooltip>
      <Tooltip position="bottomRight" content="Bottom Right Tooltip">

### Caret

The Tooltip component allows you to toggle the caret on and off using the `caret` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipCaretGroup = () => {
  return (
    <div className="flex items-center gap-8 flex-wrap">
      <Tooltip caret={true} content="This is a tooltip with caret">
        <Button className="w-28">With caret</Button>
      </Tooltip>
      <Tooltip caret={false} content="This is a tooltip without caret">
        <Button className="w-28">Without caret</Button>
      </Tooltip>
    </div>
  );
};

<TooltipCaretGroup />
</Tabs.Tab>

<Tabs.Tab>
```jsx copy
const App = () => {
  return (
    <div className="flex items-center gap-8 flex-wrap">

        <Button className="w-28">With caret</Button>
      </Tooltip>
      <Tooltip caret={false} content="This is a tooltip without caret">
        <Button className="w-28">Without caret</Button>
      </Tooltip>
    </div>
  )
};
export default App;
```
</Tabs.Tab>

### Size

The Tooltip component comes with 2 different sizes that you can change it using the `size` prop. The primary and secondary buttons can be used only in large tooltips. The width of small tooltips change according to the text length.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const TooltipSizeGroup = () => {
  return (
    <div className="flex items-end gap-3">
        <Tooltip size="sm" content="Small Tooltip">
          <Button className="w-20">Small</Button>
        </Tooltip>
        <Tooltip size="lg" content="Large Tooltip">
          <Button className="w-24">Large</Button>
        </Tooltip>
    </div>
  );
};

<TooltipSizeGroup />
</Tabs.Tab>

  <Tabs.Tab>
