# OTP Input

## Import 
    
    ```jsx
    import { VKOtpInput } from "@vivakits/react-components";
    ```

### Basic Usage
The  `VKOtpInput`  component is used to get the OTP from the user. 

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
        <VKOtpInput />
    </Tabs.Tab>   
    <Tabs.Tab>
        ```jsx copy
        import React from 'react';
        import {VKOtpInput} from "@vivakits/react-components";

        const App = ()=> {
            return(
                <VKOtpInput />
            )
        }

        export default App;
        ```
    </Tabs.Tab>
</Tabs>

### Size
The size of the OTP Input can be updated using `size` prop. The default size is `md`. The available sizes are `sm`, `md`, `lg`, `xl`. The examples below show the input with `sm` and `xl` sizes.

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
       <div className="flex items-center flex-wrap justify-around py-8">
            <VKOtpInput size="sm" />
            <VKOtpInput size="xl" />
        </div>
    </Tabs.Tab>   
    <Tabs.Tab>
        ```jsx copy
        import React from 'react';
        import {VKOtpInput} from "@vivakits/react-components";

        const App = ()=> {
            return(
                <div className="flex items-center flex-wrap justify-around py-8">
                    <VKOtpInput size="sm" />
                    <VKOtpInput size="xl" />
                </div>
            )
        }

        export default App;
        ```
    </Tabs.Tab>
</Tabs>

### Rounded
The shape of the OTP Input component can be customized using the rounded prop. The possible shapes include `none`,`default`, `xs`, `sm`, `md`, `lg`, `xl`, and `full`. The examples below display the OTP Input with the shapes `none`, `xs`, `lg`, and `full` respectively.

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
        <div className="flex items-center gap-16 flex-wrap justify-around py-8">
            <VKOtpInput />  
            <VKOtpInput rounded="xs" />       
            <VKOtpInput rounded="lg" />   
            <VKOtpInput rounded="full" />
        </div>
    </Tabs.Tab>
    <Tabs.Tab>
        ```jsx copy
        import React from 'react';
        import {VKOtpInput} from "@vivakits/react-components";

        const App = ()=> {
            return(
                <div className="flex items-center gap-16 flex-wrap justify-around py-8">
                    <VKOtpInput />  
                    <VKOtpInput rounded="xs" />

                    <VKOtpInput rounded="full" />
                </div>
            )
        }

        export default App;
        ```
    </Tabs.Tab>
</Tabs>

</Tabs>

### Submit 
The OTP Input value can be submitted using the `onSubmit` prop. The example below includes a enter button for this purpose.

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
        <VKOtpInput onSubmit={(value) => alert(value)} />
    </Tabs.Tab>
    <Tabs.Tab>
        ```jsx copy

        import {VKOtpInput} from "@vivakits/react-components";

        const App = ()=> {
            return(
                <VKOtpInput onSubmit={(value) => alert(value)} />
            )
        }

        export default App;
        ```
    </Tabs.Tab>
</Tabs>

### Custom Submit Button
The submit button can be customized using the `renderSubmitButton` prop. The example below shows the OTP Input with a custom submit button.

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>
        <VKOtpInput renderSubmitButton={(value, callback) => <Button rounded="md" size="md" variant="solid" className="!bg-primary !text-primary-foreground" onClick={() => callback(value)}>Submit</Button>} />
    </Tabs.Tab>
    <Tabs.Tab>
        ```jsx copy
        import React from 'react';
        import {VKOtpInput, Button} from "@vivakits/react-components";

        const App = ()=> {
            return(
                <VKOtpInput renderSubmitButton={(value, callback) => <Button rounded="md" size="md" variant="solid" className="!bg-primary !text-primary-foreground" onClick={() => callback(value)}>Submit</Button>} />
            )
        }

        export default App;
        ```
    </Tabs.Tab>
</Tabs>

### Group Size
The group size of the OTP Input can be changed using the `groupSize` prop. By default, the group size is the same as the length of the OTP Input. In the example below, the OTP Input is shown with a group size of `2`.

<Tabs items={['Preview', 'Code']}>
    <Tabs.Tab>

    </Tabs.Tab>

        ```jsx copy
            return(
                <VKOtpInput groupSize={2} />
            )
        }

        export default App;
        ```
    </Tabs.Tab>
</Tabs>
