### Variants

`variant` prop specifies visual styling of the chip component. The Chip component comes with 2 different variants that can be changed using the `variant` prop.

<Tabs items={['Preview', 'Code']}>

  <Tabs.Tab>

  return (
    
    <div className="flex items-center gap-3">
      <VKChip
          label="Outlined"
          variant={"outlined"}
        />
      <VKChip
          label="Filled"
          variant={"filled"}
        />
    </div>

);
};

Chip can be used with other component as well. Here is an example of chip being used with Input component.

<Tabs items={['Preview', 'Code']}>
  <Tabs.Tab>
export const ChipInsideInput = () => {
const [chips, setChips] = useState([]);

return(

<div className="m-2 border rounded-lg flex">
  {chips.map((chip, index) => (
    <VKChip
      key={`${chip}_${index}`}
      open
      color={"dark"}
      label={chip}
      size={"sm"}
      className="m-1"
      rightIcon={<CloseIcon />}
      onRightIconClick={() => setChips(chips.filter((_, i) => i !== index))}
    />
  ))}
  <Input
    size={"md"}
    rounded={"lg"}
    className="border-none"
    placeholder="Enter any word and press enter"
    onKeyUp={(e) => {
      if (e.code === "Enter" || e.code === "NumpadEnter") {
        const newChip = e.target?.value;
        setChips([...chips, newChip]);
        e.target.value = "";
      }
    }}
  />
</div>
)}

  <ChipInsideInput />
  
  </Tabs.Tab>
