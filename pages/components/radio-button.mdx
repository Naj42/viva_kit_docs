# Radio Button

The RadioButton component is a versatile and customizable UI element designed for selection among multiple options. It allows users to select one option from a predefined set. This component is part of a form or user interface where mutually exclusive selection is required.

### Import

To utilize the Radio button component, import it from `@vivakits/react-components`.

```jsx copy
import { VKRadioButton } from "@vivakits/react-components";
```


### Variant

The `variant` prop defines the visual variant of the radio button. This include styles like `circle` `square`, providing flexibility in appearance.

### Radio Group

The Radio Button can be used inside `VKGroup` component to build radio button group.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const RadioButtonGroup = () => {
  const [selectedHorizontal, setSelectedHorizontal] = useState("");
  const [selectedVertical, setSelectedVertical] = useState("");

    setSelectedHorizontal(event.target.value);

  const onChangeVerticalHandler = (event) => {

  return (
    <div className="flex flex-col items-start gap-3 flex-wrap">

            Accounting

            value={"marketing"}
            selectedValue={selectedHorizontal}
            onChange={onChangeHorizontalHandler}
          >
            Marketing

            value={"production"}
            selectedValue={selectedHorizontal}
            onChange={onChangeHorizontalHandler}

            Production

            Research
          </VKRadioButton>

        <VKTypography className="mb-2" variant="h6">Vertical Radio Button Group</VKTypography>
