# Radio Button

### Sizes

The RadioButton component comes with 3 different sizes that you can change it using the `size` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const RadioButtonSizeGroup = () => {
  return (
    <div className="flex items-end gap-3">
      <RadioButton size="sm">Small</RadioButton>
      <RadioButton size="md">Medium</RadioButton>
      <RadioButton size="lg">Large</RadioButton>
      <RadioButton size="xl">Extra Large</RadioButton>
    </div>
  );
};

<RadioButtonSizeGroup />
</Tabs.Tab>

  <Tabs.Tab>

### Colors

The RadioButton component comes with 7 different colors that you can change it using the `color` prop. It has both solid and outline variant

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const RadioButtonColorGroup = () => {
  return (
    <div className="flex items-end gap-3 flex-wrap">
      <RadioButton color="primary">Primary</RadioButton>
      <RadioButton color="secondary">Secondary</RadioButton>
      <RadioButton color="success">Success</RadioButton>
      <RadioButton color="warning">Warning</RadioButton>
      <RadioButton color="danger">Danger</RadioButton>
      <RadioButton color="dark">Dark</RadioButton>
      {/*<RadioButton color="light">Light</RadioButton>*/}
    </div>
  );
};

<RadioButtonColorGroup />
  </Tabs.Tab>

  <Tabs.Tab>
```jsx copy
<div className="flex items-end gap-3">
  <RadioButton color="primary">Primary</RadioButton>
  <RadioButton color="secondary">Secondary</RadioButton>
  <RadioButton color="success">Success</RadioButton>
  <RadioButton color="warning">Warning</RadioButton>
  <RadioButton color="danger">Danger</RadioButton>
  <RadioButton color="dark">Dark</RadioButton>
  <RadioButton color="light">Light</RadioButton>
</div>
  ```
  </Tabs.Tab>
</Tabs>
