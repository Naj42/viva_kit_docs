# Breadcrumb

Breadcrumb is a navigation aid which helps user to understand current location in the website hierarchy.

### Seperator

The `seperator` prop is an icon component or simply an icon to be displayed between breadcrumb items. This functions as path seperator.

<Tabs items={['Preview', 'Code']}>

  <Tabs.Tab>
export const BreadcrumbSeperator = () => {
  const VKBreadCrumbs = dynamic(() =>
  import('@vivakits/react-components').then(module => module.VKBreadCrumbs)
);
    const breadcrumbOptions = [
    {
      title: "Home",
      href: "/",
    },
    {
      title: "About",
      href: "/about",
     
    },
    {
      title: "Contact",
      href: "/about/contact",
    },
    {
      title: "Profile",
      href: "/about/contact/profile",
    },
  ];

return (

    <div className="">
       <VKBreadCrumbs
          separator={<ForwardSlashIcon />}
          options={breadcrumbOptions}
        />
    </div>

);
};

<BreadcrumbSeperator />
  </Tabs.Tab>

### Options

The `options` prop is an array of objects that represents the breadcrumb items. Each object here contains `title` that dsiplays the text for the breadcrumb item and `href` which contains the URL of the breadcrumb item.

### Underline
