# Highlight

### Import

<Tabs items={[]}>
    <Tabs.Tab>
        ```jsx copy
            import { VKHighlight} from "@vivakits/react-components";
        ```
    </Tabs.Tab>
</Tabs>

### Highlight string

Highlight component render the main string as children and pass the string you want to highlight using the `highlightText` prop 

<Tabs items={['Preview', 'Code']}>
<Tabs.Tab>
export const SingleString= () => {
  return (
    <VKHighlight highlightText="spotlight">
        With the Highlight component, you can spotlight words
    </VKHighlight>
  );
};

<SingleString />
</Tabs.Tab>
<Tabs.Tab>
```jsx copy
<VKHighlight highlightText="spotlight">
    With the Highlight component, you can spotlight words
</VKHighlight>
```
</Tabs.Tab>
</Tabs>

  );
