# Switch

The Switch component can be used to toggle between two states, on and off.

### Default Checked

The Switch component can be set to a default checked state using the `defaultChecked` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const SwitchDefaultCheckedGroup = () => {
  const [isSwitchChecked, setSwitchChecked] = useState(false);
  return (
    <VKSwitch defaultChecked onToggle={() => setSwitchChecked(!isSwitchChecked)}/>
  );
};

<SwitchDefaultCheckedGroup />
</Tabs.Tab>

<Tabs.Tab>
```jsx copy
<VKSwitch defaultChecked />
```
</Tabs.Tab>
</Tabs>

### On/Off Icon

The Switch component can display an icon when it is on using the `onIcon` prop and when it is off using the `offIcon`.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const SwitchIconGroup = () => {
  return(
    <VKSwitch
          offIcon={
            <img
              width="20"
              height="20"
              src="https://img.icons8.com/ios/50/sun--v1.png"
              alt="sun--v1"
            />
          }
          onIcon={
            <img
              width="20"
              height="20"
              src="https://img.icons8.com/ios/50/moon-symbol.png"
              alt="moon-symbol"
            />
          }
        />
  );
};

<SwitchIconGroup />
</Tabs.Tab>

<Tabs.Tab>
```jsx copy
<VKSwitch onIcon={<Icon />} offIcon={<Icon />}  />
```

</Tabs.Tab>
</Tabs>

### On/Off Inner Label

The Switch component can display a label when it is on using the `onInnerLabel` prop and when it is off using the `offInnerLabel` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const SwitchInnerLabelGroup = () => {
  const [vaar, isVaar] = useState(false)
  const [vaar2, isVaar2] = useState(false)
   return (
    <div className="flex items-center gap-8 flex-wrap">
    <VKSwitch offInnerLabel="Light"
          onInnerLabel="Dark" isChecked={vaar}
          onToggle={() => isVaar(!vaar)} />
    <VKSwitch
          isChecked={vaar2}
          onToggle={() => isVaar2(!vaar2)}
          offInnerLabel={
            <img
              width="20"
              height="20"
              src="https://img.icons8.com/ios/50/sun--v1.png"
              alt="sun--v1"
            />
          }
          onInnerLabel={
            <img
              width="20"
              height="20"
              src="https://img.icons8.com/ios/50/moon-symbol.png"
              alt="moon-symbol"
            />
          }
        />
        </div>
  );
};

<SwitchInnerLabelGroup />
</Tabs.Tab>

<Tabs.Tab>
```jsx copy
<VKSwitch onInnerLabel="On" offInnerLabel="Off" />
```

</Tabs.Tab>
</Tabs>
