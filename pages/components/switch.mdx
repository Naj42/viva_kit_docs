# Switch

The Switch component can be used to toggle between two states, on and off.

### Import 

To use the Switch component, import it from `@vivakits/react-components`.
  
  ```jsx
  import { VKSwitch } from "@vivakits/react-components";
  ```



### Basic Usage
There are some required props to use the Switch component. The `onToggle` and `isChecked` props are required to handle the toggle event. The `isChecked` prop is used to set the initial state of the switch. The `onToggle` prop is used to handle the toggle event.


<Tabs items={['Preview', 'Code']}>
<Tabs.Tab>
export const SwitchBasicGroup = () => {
  const [isSwitchChecked, setSwitchChecked] = useState(false);
  return (
    <VKSwitch isChecked={isSwitchChecked} onToggle={() => setSwitchChecked(!isSwitchChecked)} />
  );
};

<SwitchBasicGroup />
</Tabs.Tab>

<Tabs.Tab>
```jsx copy
import React from 'react';
import {VKSwitch} from "@vivakits/react-components";

const App = ()=> {
  const [isSwitchChecked, setSwitchChecked] = useState(false);
  
  return(
    <VKSwitch isChecked={isSwitchChecked} onToggle={() => setSwitchChecked(!isSwitchChecked)} />
  )
}

export default App;
``` 
</Tabs.Tab>
</Tabs>

### Variant

The Switch component can be styled in different ways using the `variant` prop. The Switch component comes with 3 different variants: `default`, `android`, and `iOS`.The variant will be `default` if the `variant` prop is not provided.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const SwitchVariantGroup = () => {
  const [isSwitchChecked, setSwitchChecked] = useState(false);
  const [isIOSSwitchChecked, setIOSSwitchChecked] = useState(false);
  const [isAndriodSwitchChecked, setAndriodSwitchChecked] = useState(false);

  return (
    <div className="flex justify-around items-center flex-wrap py-8">
      <VKSwitch isChecked={isSwitchChecked} onToggle={() => setSwitchChecked(!isSwitchChecked)} label="Default" />
      <VKSwitch variant="iOS" isChecked={isIOSSwitchChecked} onToggle={() => setIOSSwitchChecked(!isIOSSwitchChecked)} label="iOS" />
      <VKSwitch variant="android" isChecked={isAndriodSwitchChecked} onToggle={() => setAndriodSwitchChecked(!isAndriodSwitchChecked)} label="Android" />
    </div>
  );
};

<SwitchVariantGroup />
</Tabs.Tab>

### Label Placement

The Switch component can place the label in different positions using the `labelPlacement` prop. The label can be placed on the `left`, `right`, `top`, `bottom`, `topStart`,`topEnd`, `bottomStart` and `bottomEnd` of the switch. Few examples are given below.

<Tabs items={['Preview', 'Code']}>
<Tabs.Tab>

  const [isLeftSwitchChecked, setLeftSwitchChecked] = useState(false);
  const [isTopSwitchChecked, setTopSwitchChecked] = useState(false);
  const [isTopEndSwitchChecked, setTopEndSwitchChecked] = useState(false);
  const [isBottomStartSwitchChecked, setBottomEndSwitchChecked] = useState(false);

  return (
    <div className="flex items-center flex-wrap justify-around py-8">
      <VKSwitch label="Left" labelPlacement="left" isChecked={isLeftSwitchChecked} onToggle={() => setLeftSwitchChecked(!isLeftSwitchChecked)} />
      <VKSwitch label="Top" labelPlacement="top" isChecked={isTopSwitchChecked} onToggle={() => setTopSwitchChecked(!isTopSwitchChecked)} />
      <VKSwitch label="Top End" labelPlacement="topEnd" isChecked={isTopEndSwitchChecked} onToggle={() => setTopEndSwitchChecked(!isTopEndSwitchChecked)} />
      <VKSwitch label="Bottom Start" labelPlacement="bottomStart" isChecked={isBottomStartSwitchChecked} onToggle={() => setBottomEndSwitchChecked(!isBottomStartSwitchChecked)} />

    </div>
  );
};
