# Input

### Variants

The Input component comes with 2 different variants that you can change it using the `variant` prop.

<Tabs items={['Preview', 'Code']}>

  <Tabs.Tab>
export const InputVariantGroup = () => {
  return (
    <div className="flex items-center gap-3">
        <Input label="Outlined:" variant="outline" />
        <Input label="Solid:" variant="solid" className="!bg-gray-300" />        
    </div>
  );
};

<InputVariantGroup />
  </Tabs.Tab>

  <Tabs.Tab>
```jsx copy
    <div className="flex items-center gap-3">
        <Input label="Outlined input variant" variant="outline" />
        <Input label="Solid input variant" variant="solid" className="!bg-gray-300" />
    </div>
```
  </Tabs.Tab>
</Tabs>

### Icon

An icon can be append or prepend using `icon` prop. The position of the icon can be specified by the `iconPosition` prop.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const InputIconGroup = () => {
  return (
    <div className="flex items-end gap-3">
      <Input icon={<FaSearch />} placeholder="Search" />
      <Input iconPosition="end" icon={<FaSearch />} placeholder="Search" />
    </div>
  );
};

<InputIconGroup />
</Tabs.Tab>

<Tabs.Tab>
  ```jsx copy
  <div className="flex items-end gap-3">
    <Input icon={<FaSearch />} placeholder="Search" />
    <Input iconPosition="end" icon={<FaSearch />} placeholder="Search" />
  </div>
  ```
</Tabs.Tab>

</Tabs>

### Error

Conditional error handling for input can be done via `hasError` props. Error message can also be modified via `errorMessage` props.

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const InputErrorGroup = () => {
  return (
    <div className="flex items-end gap-3">
      <Input icon={<FaEdit />} placeholder="Error variant" hasError={true} />
      <Input icon={<FaEdit />} placeholder="Error variant" hasError={true} errorMessage="Custom error message" />
    </div>
  );
};

<InputErrorGroup />
</Tabs.Tab>

<Tabs.Tab>
  ```jsx copy
  <div className="flex items-end gap-3">
    <Input icon={<FaEdit />} placeholder="Error variant" hasError={true} />
    <Input icon={<FaEdit />} placeholder="Error variant" hasError={true} errorMessage="Custom error message" />
  </div>
  ```
</Tabs.Tab>

</Tabs>
