# Label

### isRequired

This is a boolean prop indicating whether the field associated with this label is required or not. This can be controlled using `isRequired` prop.

<Tabs items={['Preview', 'Code']}>
<Tabs.Tab>
export const LabelIsRequired = () => {
  return (
    <div className="flex items-center gap-3">
    <VKLabel isRequired={true}>Name:</VKLabel>
    <input placeholder="Enter name" className="border border-gray-600 rounded-md w-52 h-9 p-1"/>
    </div>
);
};

<LabelIsRequired />
  </Tabs.Tab>

  <Tabs.Tab>

```jsx copy
    </div>
```
    </Tabs.Tab>
 </Tabs>

### isDisabled

This is a boolean prop that specifies whether the label should appear as disabled or not. It can be controlled using `isDisabled`
prop
