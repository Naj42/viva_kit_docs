# Button

### Variants

The Button component comes with 3 different variants that you can change it using the `variant` prop.

<Tabs items={['Preview', 'Code']}>

  <Tabs.Tab>
export const ButtonVariantGroup = () => {
  return (
    <div className="flex items-center gap-3">
        <Button variant="solid" className="!bg-primary !text-primary-foreground" >Solid</Button>
        <Button variant="outline">Outline</Button>
        <Button variant="light" className="!bg-primary-light !text-primary">Light</Button>
    </div>
  );
};

<ButtonVariantGroup />
  </Tabs.Tab>

  <Tabs.Tab>
```jsx copy
<div className="flex items-center gap-3">
    <Button variant="solid" >Solid</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="light">Light</Button>
</div>
  ```
  </Tabs.Tab>
</Tabs>

### Colors

The Button component comes with 5 different colors that you can change it using the `color` prop. It has both solid and outline variant

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const ButtonColorGroup = () => {
  return (
    <div className="flex items-end gap-3 flex-wrap">
        <Button className="!bg-primary !text-primary-foreground">Primary Solid</Button>
        <Button variant="outline" color="primary">Primary Outline</Button>
        <Button className="!bg-secondary !text-secondary-foreground">Secondary Solid</Button>
        <Button variant="outline" color="secondary">Secondary Outline</Button>
        <Button className="!bg-success !text-success-foreground">Success Solid</Button>
        <Button variant="outline" color="success">Success Outline</Button>
        <Button className="!bg-warning !text-warning-foreground">Warning Solid</Button>
        <Button variant="outline" color="warning">Warning Outline</Button>
        <Button className="!bg-danger !text-danger-foreground">Danger Solid</Button>
        <Button variant="outline" color="danger">Danger Outline</Button>
        
    </div>
  );
};

<ButtonColorGroup />
  </Tabs.Tab>

  <Tabs.Tab>

<div className="flex items-end gap-3">

  <Button variant="outline" color="secondary">Secondary Outline</Button>
  <Button color="success">Success Solid</Button>
  <Button variant="outline" color="success">Success Outline</Button>
  <Button color="warning">Warning Solid</Button>
  <Button variant="outline" color="warning">Warning Outline</Button>
  <Button color="danger">Danger Solid</Button>
  <Button variant="outline" color="danger">Danger Outline</Button>

### Rounded

The shape of the Button component can be updated using `rounded` prop

<Tabs items={['Preview', 'Code']}>

<Tabs.Tab>
export const ButtonRoundedGroup = () => {
  return (
    <div className="flex items-end gap-3">
        <Button rounded="none" className="btn">None</Button>
        <Button rounded="default" className="btn">Default</Button>
        <Button rounded="sm" className="btn">Small</Button>
        <Button rounded="md" className="btn">Medium</Button>
        <Button rounded="lg" className="btn">Large</Button>
        <Button rounded="full" className="btn">Full</Button>
    </div>
  );
};

<ButtonRoundedGroup />
</Tabs.Tab>

<Tabs.Tab>
  ```jsx copy
  <div className="flex items-end gap-3">
    <Button rounded="none">Nome</Button>
    <Button rounded="default">Default</Button>
    <Button rounded="sm">Small</Button>
    <Button rounded="md">Medium</Button>
    <Button rounded="lg">Large</Button>
    <Button rounded="full">Full</Button>
  </div>
  ```
</Tabs.Tab>
